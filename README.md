## Installation

### Local Development

1. Clone the repository
```bash
git clone https://github.com/genesisValue/anastasis-website.git
cd anastasis-website
```

2. Open in your browser
```bash
open index.html
# or
start index.html
# or use any local server
python -m http.server 8000
```

### Deployment

#### Vercel (Recommended)

1. Push to GitHub
2. Connect repository to [Vercel](https://vercel.com)
3. Auto-deploys on every push
4. Custom domain ready

#### GitHub Pages

1. Push to GitHub
2. Go to Settings → Pages
3. Select main branch as source
4. Done! Access via `https://username.github.io/anastasis-website`
